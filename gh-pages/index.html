What is FromJS?

FromJS is a developer tool that helps you understand how a JavaScript app works.

You can select an HTML element in the page you're inspecting to view where its content came from. In the screenshot you can see that the string "Hello" originally came from localStorage data.

Other possible string sources include string literals in the app's JS code, as well as HTML served on the initial page load.

Does it work for all sites?

FromJS won't work on most sites, yet. There's a lot of work to be done to support more browser functionality. Large sites currently also take a long time to load and use up large amounts of memory.

Can I try it anyway?

Yes, just install the FromJS Chrome Extension. The Github readme lists some sites where FromJS works fairly well.
