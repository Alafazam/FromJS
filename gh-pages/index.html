What is FromJS?

FromJS is a developer tool that helps you understand how an app works.

You can select an HTML element in the inspected page and view where in the code it originated. In this screenshot I've selected a label tag on the page, specifically I'm tracing the origin of the "B" character.

At the bottom you can see that it was added to the DOM through an innerHTML assignment. At the top you can see that the "Buy milk" string originally came from localStorage data.

Does it work for all sites?

FromJS won't work on most sites, yet. There's a lot of work to be done to support more browser functionality. Large sites currently also take a long time to load and use up large amounts of memory.

Can I try it anyway?

Yes, just install the FromJS Chrome Extension. The Github readme lists some sites where FromJS works fairly well.
