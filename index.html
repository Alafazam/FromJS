What is FromJS?

FromJS is a dynamic analysis tool that helps you understand how an app works.

You can select an HTML element in the analysed page and view where in the code it originiated. In this screenshot I've selected a label tag on the page, specifically I'm tracing the origin of the "B" character.

At the bottom you can see that it was added to the DOM through an innerHTML assignment. At the top you can see that the "Buy milk" string originally came from localStorage data.

Is it ready yet?

Not yet, at least not for anything complex. I've written the code so it has the functionality required for the TodoMVC demo. FromJS will work if you use a subset of this functionality, but chances are your app won't even render.
